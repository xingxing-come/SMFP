Article: SMFP: A Self-Supervised Multimodal Molecular Feature Fusion Method with Bidirectional Cross-Attention for Molecular Property Prediction

Model Overview
SMFP is a self-supervised multimodal framework that integrates different molecular representations through a bidirectional cross-attention mechanism.
It captures both complementary and interactive information across modalities to enhance molecular property prediction performance.
